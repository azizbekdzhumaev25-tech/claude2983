# Modern Landing Site 🎨

Красивый современный одностраничный сайт с адаптивным дизайном.

## 📁 Структура проекта

```
claude2983/
├── index.html          # Главная HTML страница
├── README.md           # Документация
└── src/
    ├── css/
    │   └── style.css   # Основные стили
    ├── js/
    │   └── main.js     # Основной JavaScript
    └── images/         # Изображения (пусто)
```

## ✨ Возможности

- ✅ **Современный дизайн** - Минималистичный и элегантный
- ✅ **Адаптивность** - Работает на всех устройствах (мобильные, планшеты, десктопы)
- ✅ **Анимации** - Плавные анимации при скролле и наведении
- ✅ **Эффект печатающего текста** - Динамический текст в hero секции
- ✅ **Счетчики статистики** - Анимированные счетчики
- ✅ **Валидация формы** - Проверка полей формы обратной связи
- ✅ **Навигация** - Плавный скролл и активные ссылки
- ✅ **Темная/светлая тема** - Готов к расширению

## 🚀 Как запустить

### Локально
1. Клонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в браузере

### С использованием сервера
```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен)
npx serve

# PHP
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000`

## 🎯 Кастомизация

### Изменение цветов
Откройте `src/css/style.css` и измените CSS переменные в начале файла:
```css
:root {
    --primary-color: #6c5ce7;
    --secondary-color: #00cec9;
    --accent-color: #fd79a8;
    /* ... */
}
```

### Изменение текста
Откройте `index.html` и измените текст в соответствующих секциях.

### Добавление изображений
Поместите изображения в папку `src/images/` и обновите пути в HTML.

## 📱 Адаптивность

Сайт полностью адаптивен и работает на:
- Десктопах (1200px+)
- Планшетах (768px - 1199px)
